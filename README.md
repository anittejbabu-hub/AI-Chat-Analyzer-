# 🤖 AI Smart Chat Analyzer

## 📌 Project Overview
AI Smart Chat Analyzer is a Python-based application that analyzes chat conversations and provides meaningful insights such as sentiment analysis and keyword extraction.

This project demonstrates the application of Natural Language Processing (NLP) and Machine Learning concepts in real-world chat data.

---

## 🚀 Features
- ✅ Chat sentiment analysis (Positive / Negative / Neutral)
- ✅ Text preprocessing
- ✅ Keyword extraction
- ✅ User-friendly interface
- ✅ Real-time chat analysis

---

## 🛠️ Tech Stack
- Python
- HTML (Frontend)
- NLP
- Machine Learning
- Pandas
- NLTK / Scikit-learn (if used)
